# demo_exam_first_module
Assignment from the demonstration exam for the specialty "Information technology testing specialist

Разработайте набор модульных тестов и оформите тест-кейсы на
добавление нового пользователя в систему. Используйте технологию TDD:
сначала напишите тесты, затем реализуйте код.
Функциональность, подлежащая тестированию: добавление пользователя в
систему через модальное окно «Добавить пользователя».
Поля для ввода в форме:
- Полное имя — строка, не более 40 символов, без цифр и спецсимволов;
- Дата рождения — пользователь должен быть старше 16 лет на момент
внесения записи;
- Электронная почта — до 64 символов, в формате user@domain.zone;
- Телефон — формат строго +7-XXX-XXX-XX-XX;
- ИНН — 12 цифр, только для граждан РФ, не должен начинаться с 00;
- Номер паспорта — 10 цифр, допустимые серии: 01–99, контроль на
уникальность в пределах базы.
Модальное окно содержит:
- 6 полей для ввода;
- кнопки "Сохранить" и "Отмена".35
Этапы выполнения:
1. Создайте проект с модульными тестами в выбранной вами среде
разработки.
2. Напишите модульные тесты:
- используйте классы эквивалентности и граничные значения;
- не менее 5 наборов данных (в том числе минимум 2 негативных сценария);
- допускается использование средств автогенерации данных.
